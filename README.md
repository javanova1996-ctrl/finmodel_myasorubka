# finmodel_myasorubka

Интерактивная финансовая модель на базе Excel-файла `ОПиУ_2026_АНАЛИТИКА.xlsx`.

## Что внутри

- факт за **январь–март 2026** из исходной Excel-модели;
- сценарный прогноз на **апрель–декабрь 2026**;
- анализ по подразделениям:
  - Халтурина
  - Целинная
  - Газонная
  - Снайперов
  - Цех
- ключевые метрики:
  - выручка
  - переменные затраты
  - постоянные затраты
  - валовая прибыль
  - чистая прибыль
  - валовая и чистая маржа

## Логика модели

Приложение читает исходный Excel-файл и берет фактическую базу из листов:

- `СВОД_01` — январь
- `СВОД-02` — февраль
- `СВОД-03` — март

Для прогноза используются:

- последняя фактическая выручка как база;
- средняя историческая доля переменных затрат;
- последние фактические постоянные затраты как база;
- пользовательские сценарные корректировки:
  - ежемесячный рост выручки;
  - изменение доли переменных затрат;
  - рост постоянных затрат;
  - индивидуальные поправки по каждому подразделению.

## Запуск локально

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

## Публикация на GitHub

1. Создай новый репозиторий на GitHub.
2. Загрузи в него содержимое этой папки.
3. Сделай первый коммит:

```bash
git init
git add .
git commit -m "Initial commit: interactive finmodel"
git branch -M main
git remote add origin <URL_твоего_репозитория>
git push -u origin main
```

## Публикация в Streamlit Community Cloud

1. Запушь проект в GitHub.
2. Открой Streamlit Community Cloud.
3. Выбери репозиторий и файл `app.py`.
4. Нажми Deploy.
